# Design System

모던하고 확장 가능한 디자인 시스템입니다.

## 개요

이 디자인 시스템은 일관된 사용자 경험을 제공하기 위한 재사용 가능한 컴포넌트와 디자인 토큰의 모음입니다.

### 주요 특징

- **일관성**: 모든 컴포넌트가 동일한 디자인 원칙을 따릅니다
- **접근성**: WCAG 2.1 AA 기준을 충족합니다
- **확장성**: 쉽게 커스터마이징하고 확장할 수 있습니다
- **타입 안정성**: TypeScript로 작성되어 타입 안정성을 보장합니다

## 디자인 토큰

디자인 토큰은 디자인 시스템의 기본 빌딩 블록입니다:

- **Colors**: 색상 팔레트 및 시맨틱 컬러
- **Typography**: 폰트 크기, 두께, 행간
- **Spacing**: 일관된 여백 및 간격
- **Shadows**: 깊이 표현을 위한 그림자
- **Radius**: 모서리 둥글기

## 컴포넌트

다양한 UI 요구사항을 충족하는 재사용 가능한 컴포넌트를 제공합니다:

### 기본 컴포넌트
- Button, Input, TextArea, Select, Checkbox, Radio

### 피드백 컴포넌트
- Alert, Toast, Spinner, Modal

### 레이아웃 컴포넌트
- Card, Stack, Divider

### 타이포그래피
- Typography, Badge

## 사용 방법

```typescript
import { Button, Alert } from '@design-system/components';
import { colors, spacing } from '@design-system/tokens';

function App() {
  return (
    <div>
      <Alert variant="info">
        디자인 시스템에 오신 것을 환영합니다!
      </Alert>
      <Button variant="primary">시작하기</Button>
    </div>
  );
}
```
